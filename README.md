# Book-store-RestAPI
This is a project built with Express, MongoDB, Mongoose and using MVC architecture

# Run the service
$ npm start

# API Endpoints:
# Book Routes 
- Create a new book
POST |      /v1/book

- Get all books
GET |       v1/book

- Get one book
GET |       v1/book/:id

- Delete one book
DELETE |       v1/book/:id

- Update one book
PUT |       v1/book/:id

# Author Routes
- Create a new author
POST |      /v1/author

- Get all authors
GET |       v1/author

- Get one author
GET |       v1/author/:id

- Delete one author
DELETE |       v1/author/:id

- Update one author
PUT |       v1/author/:id
